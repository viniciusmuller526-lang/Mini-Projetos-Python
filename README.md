Bem-vindo a minha coleção de mini-projetos!

Este repositório foi criado durante o meu **1º Semestre de Sistemas de Informação** para consolidar o aprendizado de lógica de programação, estruturas de dados básicas e modularização em Python, aplicando os conceitos em cenários reais e interativos.

---

## 🚀 Os Projetos

Aqui está a lista de aplicações desenvolvidas e as principais lógicas aplicadas em cada uma:

### 🧮 1. Calculadora Modular Interativa
Uma calculadora baseada em terminal utilizando funções para cada operação matemática.
* **Destaques Técnicos:** Modularização avançada (`def`), tratamento de erro personalizado para evitar divisão por zero utilizando retornos nulos (`None`) e estrutura de menu persistente.

### 🛒 2. Gerenciador de Lista de Compras
Um sistema CRUD (Criar, Ler, Deletar) básico para gerenciamento de listas.
* **Destaques Técnicos:** Manipulação dinâmica de arrays (Listas em Python) com comandos `.append()` e `.remove()`. Código blindado contra quebras (*crashes*) através da checagem prévia de existência do item.

### 👾 3. Mini Tamagotchi (Pet Virtual)
Simulador de bichinho virtual onde o jogador precisa equilibrar os níveis de fome e energia do pet.
* **Destaques Técnicos:** Controle de estados interligados, desempacotamento de tuplas (múltiplos retornos em uma única função) e lógica de *clamping* (limitação de bordas) para garantir que os status nunca passem do máximo (10) ou do mínimo (0).

---

## 🛠️ Competências Desenvolvidas

Ao construir esses projetos, pratiquei e dominei:
* **Estruturas de Controle:** Laços de repetição (`while`, `for`) e tomadas de decisão.
* **Modularização:** Criação de código limpo, reaproveitável e encapsulado em funções.
* **Gerenciamento de Estados:** Passagem de parâmetros e escopos de variáveis dentro do fluxo do programa.
